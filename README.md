# Hospitalweb
一个医院门户网站的前端设计与实现。主要使用了Bootstrap，适配移动端和PC端。

# 包含模块
*首页
*登陆注册
*医院概括
*医疗团队
*新闻中心
*就医指南
*健康教育
*科室医生搜索
*个人中心
*我要预约
*最近预约记录
*基本信息
*修改密码
